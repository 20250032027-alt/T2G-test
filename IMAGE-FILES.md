# T2G Image Files Guide

All images go in the `assets/img/` folder.
When you add an image with the exact filename listed below and redeploy,
it will appear automatically on the site.

## Product Images
The filename for each product is set in Dev Mode (Alt+1).
Default filenames match the product ID. Examples:

| Default Filename | Product |
|---|---|
| `popcorn.png`        | Popcorn |
| `coco-balsamic.png`  | Coco Balsamic Vinegar |
| `coco-vinegar.png`   | Coco Vinegar |
| `coco-seasoning.png` | Coco Seasoning |
| `virgin-coco-oil.png`| Virgin Coco Oil |
| `coco-syrup.png`     | Coco Syrup |
| `coco-jam.png`       | Coco Jam |
| `coco-sugar.png`     | Coco Sugar |

To change the filename for any product:
1. Press Alt+1 to open Dev Mode
2. Click EDIT on the product
3. Change the "Image Filename" field (e.g. `my-photo.jpg`)
4. Save
5. Put the matching file in `assets/img/`
6. Redeploy

## Tips
- JPG for photos (smaller file size), PNG for logos and certifications (transparent backgrounds)
- Recommended photo quality: 80-85% JPG compression
- Keep product images square (1:1 ratio) for consistent card display
- Hero images: wide and landscape (16:6 ratio works best)
